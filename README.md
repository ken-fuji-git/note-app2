# プロダクト名
おかげ犬珍道中

# デプロイURL
https://fujiwarakenta.sakura.ne.jp/t/login

## 使い方
- ログインします。
- 犬の写真、名前、性別、年齢、犬種、体高、性格（自由記入）を入力します。
- 「更新して出発画面へ」を押します。
- お願い事を入力します。ステキな願いを託しましょう。
- 出発する場所を選択します。通知が出てきたら許可をしてください。
- よければ「ここから出発」を押します。
- 道のりと推定日数が出てきます。
- 「珍道中を見る」を押します。
- 読み込みに時間がかかりますが、がんばって進む道のりが確認できるはずです。
- どんな物語が待っているのでしょう？楽しみに読んでみてください。
- 無事にお参りが済むと記念の木札画像が出てきます。是非シェアしてください。
- 今回のストーリーは珍道中日記からいつでも確認できます。

## プロダクトの見どころ
- おかげ犬。それは江戸時代に病気や高齢などで伊勢神宮に参拝できない飼い主に代わり、首に代参の金品を下げて一人（一匹）で参拝した忠犬のことである！
- お伊勢参りってありますよね？一生に一度の贅沢だった時代に、江戸や各地方から人の代わりに犬をお遣いに出して伊勢参りさせていた事例があったそうです。色んな人の助けを借りて数ヶ月かけてちゃんと役目を果たして帰ってきたらしいですよ。色んな善意のリレーのおかげあっての事ですが、とってもえらいですね。今の時代ではほぼ実現不可能になってしまった昔の文化に思いを馳せつつ、もし、うちの仔をおかげ参りに出したらどんな展開・物語があるんだろう？という興味から作りました。
